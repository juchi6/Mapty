# Mapty – Workout Tracker

Mapty is a location-based workout tracking application that allows users to log and visualize their workouts directly on an interactive map. It uses geolocation and browser storage to provide a smooth and persistent experience.

FEATURES

Interactive Map Logging  
Click on the map to add a new workout and automatically capture location coordinates.

Geolocation Support  
Detects and displays your current location for a more personalized experience.

Workout Input Forms  
Running: distance, time, pace, steps per minute  
Cycling: distance, time, speed, elevation gain  

Workout List View  
Displays all logged workouts in a structured list.

Map Visualization  
Shows all workouts directly on the map for easy tracking.

Local Storage Persistence  
Saves workout data in the browser using local storage.

Automatic Data Reload  
Loads and displays saved workouts when the page is refreshed.

TECH STACK

JavaScript (ES6+)  
HTML5 and CSS3  
Geolocation API  
Local Storage API  
Map library (e.g. Leaflet.js)

HOW IT WORKS

1. Allow location access to display your current position  
2. Click anywhere on the map to log a workout  
3. Fill in workout details (running or cycling)  
4. View workouts in the list and on the map  
5. Data is saved automatically in your browser
